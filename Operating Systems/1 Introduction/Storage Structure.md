# Storage Structure

## Main memory

- Random access
- Typically volatile (easily evaporated)
- Typically DRAM

## Secondary Storage

- Nonvolatile
- HDD
- NVM

## Definitions and Notation

- Kilobyte - 1024 bytes
- Megabyte - 1024^2 bytes
- Gigabyte - 1024^3 bytes
- Terabyte - 1024^4 bytes
- Petabyte - 1024^5 bytes
